Two libraries are mandatory for building this projekt:
	1) org.osgi.core-4.2.0.jar
	2) org.osgi.enterprise-4.2.0.jar
	
These jars should be copied or linked from the target applicationserver.

In case of jboss-as-7.1.1.Final these jars can bo found here:
	1) <JBoss installation path>\standalone\lib\ext\org.osgi.core-4.2.0.jar
	2) <JBoss installation path>\bundles\org\osgi\enterprise\main\org.osgi.enterprise-4.2.0.jar
